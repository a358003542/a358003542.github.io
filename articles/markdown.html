github flavored markdown

gfm的官方文档在 这里 。github flavored markdown 也就是github的markdown方言，其主要区别有:

  1. 下划线直接为下划线。

  2. URL 直接输入，比如 http://www.google.com ，其将直接转化成为链接。这个只是一个新功能支持，原来的链接插入方式一样有效。

  3. 删除线 用:

    ~~删除线~~
